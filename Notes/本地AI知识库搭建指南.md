## 核心功能

### 1. 本地知识检索
Hermes Agent 可以扫描整个 Obsidian vault，建立语义索引，支持自然语言查询：
- “我之前写的关于 Docker 的笔记在哪里？”
- “找出所有提到 ‘AI Agent’ 的笔记”
- “总结一下我在 2025 年关于投资的思考”

### 2. 智能笔记创建
当在 Hermes Agent 中讨论话题时，可自动创建或更新 Obsidian 笔记：
- 讨论完技术方案后，自动生成技术文档笔记
- 阅读完文章后，自动生成摘要和关键点笔记
- 开会讨论后，自动生成会议纪要

### 3. 跨笔记关联
系统分析笔记间关联性，自动建议相关笔记链接，发现隐藏联系（优于 Obsidian 原生反向链接）。

### 4. 知识库维护
自动执行：
- 整理杂乱笔记
- 修复损坏链接
- 优化笔记结构
- 清理重复内容

### 5. 安全性保证
所有操作完全本地化：
- 无需上传笔记到任何服务器
- 无需担心隐私泄露
- 完全掌控知识资产

## 知识入库流程 (hermes-knowledge-pipeline Skill)

当用户说「Memo:」开头的话时：
1. 提取关键信息
2. 写入 `Inbox/YYYY-MM-DD.md`（按日期归档）
3. 执行 `git add`、`commit`、`push`
4. 回复确认

提交规范：`[inbox] 简短描述`

## 目录结构
- `Inbox/` — 随手记录，还没整理的
- `Tasks/` — 待办任务
- `Notes/` — 整理好的笔记（本文档所在位置）
- `Resources/` — 链接和资源收藏
